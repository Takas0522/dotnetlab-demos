# テストの実行の仕方

1. `api`のディレクトリのAzure Functionsコードをデバッグ実行
2. `front`ディレクトリで`npm ci`
3. `front`ディレクトリで`npm run start`
4. `e2e/ts`ディレクトリでテスト実行できるようになる
  * デバッグ実行しても良し。`nom run test`でテスト実行してもよし
